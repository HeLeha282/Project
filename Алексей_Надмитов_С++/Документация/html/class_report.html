Подробное описание

Класс для сбора и отображения статистики сканирования.

Класс предоставляет потокобезопасные методы для накопления статистики в процессе сканирования файлов и вывода итогового отчета. Все счетчики реализованы как атомарные переменные для безопасного использования в многопоточной среде.
