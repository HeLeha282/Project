Подробное описание

Основной класс приложения-сканера.

Класс предоставляет точку входа для запуска процесса сканирования файловой системы на наличие вредоносных файлов с использованием базы хэшей и логирования результатов.

Методы

◆ run()

Report Application::run ( const std::string & base,
const std::string & log,
const std::string & path )

Запускает процесс сканирования файловой системы.

Метод инициализирует необходимые компоненты (базу данных, логгер, сканер), запускает рекурсивное сканирование указанной директории и возвращает отчёт о выполнении.

Аргументы
baseПуть к CSV-файлу с базой вредоносных хэшей.
logПуть к файлу для записи логов.
pathПуть к корневой директории для сканирования.
Возвращает
Report Объект с результатами сканирования (количество файлов, ошибок, время и т.д.).
Заметки
Возвращаемый объект Report создаётся непосредственно в return-выражении с применением RVO (Return Value Optimization) для избежания лишнего копирования.
Исключения
std::runtime_errorЕсли не удалось открыть файлы базы или лога, или если указанная директория недоступна.
