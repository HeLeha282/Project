\doxysubsection{Подробное описание}
Класс, предоставляющий статические утилиты для обработки строк. 

Этот класс содержит набор статических методов, выполняющих распространённые операции со строками. В текущей реализации предоставляет функцию удаления пробельных символов.

{\footnotesize\ttfamily \label{class_string_utils_ab4cbc55bad618b2b73c89d4c3e63230d} 
std\+::string String\+Utils\+::trim (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{str}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}



Удаляет пробельные символы в начале и конце строки. 


\begin{DoxyParams}{Аргументы}
{\em str} & Входная строка, которую необходимо обработать. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Возвращает}
Обрезанная копия строки без пробельных символов по краям. Если строка состоит только из пробельных символов, возвращает пустую строку.
\end{DoxyReturn}
Пробельными символами считаются\+: пробел, табуляция ({\ttfamily \textbackslash{}t}), новая строка ({\ttfamily \textbackslash{}n}), возврат каретки ({\ttfamily \textbackslash{}r}), прогон страницы ({\ttfamily \textbackslash{}f}) и вертикальная табуляция ({\ttfamily \textbackslash{}v}).

\begin{DoxyNote}{Заметки}
Это дополнительная служебная функция, используемая для предобработки данных, например, при чтении строк из CSV-\/файла.
\end{DoxyNote}
