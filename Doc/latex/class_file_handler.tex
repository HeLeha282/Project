\doxysubsection{Подробное описание}
Класс для обработки отдельных файлов. 

Отвечает за вычисление хэша файла, проверку его в базе вредоносных хэшей и обновление статистики в отчете. Каждый экземпляр используется для обработки одного файла.

Обрабатывает один файл\+: вычисляет хэш и проверяет на вредоносность. 

Метод выполняет последовательно\+:
\begin{DoxyEnumerate}
\item Увеличивает счетчик обработанных файлов
\item Открывает файл в бинарном режиме
\item Вычисляет MD5-\/хэш содержимого
\item Проверяет хэш в базе данных
\item Обновляет статистику и логирует результат
\end{DoxyEnumerate}


\begin{DoxyParams}{Аргументы}
{\em file\+Path} & Полный путь к обрабатываемому файлу.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Заметки}
При ошибке открытия файла увеличивается счетчик ошибок и запись добавляется в лог. 

Файлы большого размера обрабатываются потоково без загрузки в память целиком. 
\end{DoxyNote}
