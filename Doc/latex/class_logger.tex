\doxysubsection{Подробное описание}
Класс для потокобезопасного логирования в файл. 

Класс предоставляет методы для записи различных типов сообщений в лог-\/файл\+:
\begin{DoxyItemize}
\item Обнаружение вредоносных файлов
\item Ошибки при работе утилиты
\item Информационные сообщения
\end{DoxyItemize}

Все операции записи защищены мьютексом для корректной работы в многопоточной среде. Формат записи\+: \mbox{[}ТИП\mbox{]} ВРЕМЯ \texorpdfstring{$\vert$}{|} Дополнительная информация \begin{Desc}
