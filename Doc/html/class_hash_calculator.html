Подробное описание

Класс, предоставляющий статический метод для вычисления MD5-хеша файла.

Класс содержит единственный статический метод, который вычисляет хеш-сумму MD5 для содержимого переданного файлового потока. Работает с файлами любого размера, читая данные блоками.
