Подробное описание

Класс для потокобезопасного логирования в файл.

Класс предоставляет методы для записи различных типов сообщений в лог-файл:

  • Обнаружение вредоносных файлов
  • Ошибки при работе утилиты
  • Информационные сообщения

Все операции записи защищены мьютексом для корректной работы в многопоточной среде. Формат записи: [ТИП] ВРЕМЯ | Дополнительная информация
